
# QA-java-diplom-1

Тестирование программы, которая помогает заказать бургер в Stellar Burgers.


## Technologies
- Java 11

#### Properties
- maven 11

#### Dependencies
- junit:junit:4.13.2(test)
- mockito-core 3.12.4
- com.github.javafaker:javafaker:1.0.2

#### Plugins
- maven-compiler-plugin 3.8.1
- jacoco-maven-plugin 0.8.7


## Running Tests

To run tests, run the following command

```bash
  mvn verify
```
